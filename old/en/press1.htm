The step towards absolute security

Safety-Lab (www.safety-lab.com), a company known for its professional solutions in the sphere of network security, has just presented the latest sixth version of its network scanning and auditing tool called Shadow Security Scanner (SSS). SSS is a result of more than eight-year research, significant consulting experience and novel approaches to software architecture. The new product thus makes up a part of security solutions family which also incorporates such tools as Web Analyzer, Enterprise Web Firewall and Database Scanner.

In so doing the company has definitely managed to outperform not only the existing market leaders but its own hit - the previous version of SSS that received numerous awards for speed and scanning quality.

The scanner represents the new generation of powerful multifunctional network software. It enables network administrators and analysts as well as ordinary network users to promptly trace down and fix network exposures and vulnerabilities. The implemented functionality is really astonishing - the number of vulnerability tests exceeds 2, 000.

The customer support is seen nowadays not just as a bonus, but as a must. With that in mind, Safety-Lab has accompanied its new product with the direct access to its Internet Security Expert service and a daily-updated Download Zone.

The format of the SSS check report may be easily suited to that accepted in your company - the scanner now supports XML, RTF, CHM (compiled HTML) and PDF formats as well as some traditional ones.

The users of the previous versions can not but notice the completely redesigned interface. It became more transparent, user-friendly and task-oriented providing a simple one-click access to main functions. The explanations for each element also make the work safe, straightforward and swift.

Other features include COM, DCOM, ActiveX, OLE support, significant flexibility in rules editing, comprehensive LDAP and proxy server audit, plentiful checks like those for Cisco, DoS attack withstand and password bruteforce resistance. The speed factor has also been greatly increased due to simultaneous multiple network scanning that allows up to 10 hosts per session.

As a result Safety-Lab has made the everyday network security maintenance easy having supplied the network specialists with the robust up-to-date scanning and auditing tool. The latest version is available for trial download from www.safety-lab.com\download.
